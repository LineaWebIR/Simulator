Hello, this project is with a two-person team in Iran. This project is a drone simulator for detecting natural disasters that have occurred in Iran. We are working on a physical project whose image is in the Documents folder. The simulator codes are in the Simulator folder. A presentation of natural disasters that are mentioned in the simulator, such as the Tabas earthquake, is available in both PowerPoint and PDF formats in the Documents folder + a photo of the physical project. And a video introducing and operating the simulator is available in both Persian and English in the Presentation folder. The English video has a pause in some situations because we did not use artificial intelligence for translation. There is also a teaser made with artificial intelligence for introduction. This simulator uses the Anthropic API in some situations and is web-based. Because our computer system is weak, the graphics of the simulator are weak and low so that we do not have any problems running it. Thanks (:
